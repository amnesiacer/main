## 帮扶项目收益仪表板修复

- **问题**：`dashboard.html` 无法显示数据，点击项目无响应
- **根因**：`pam.py` 的 `_build_dashboard_html` 函数中，`renderProjects()` 生成的 onclick 处理器有 JS 语法错误
  - 原始代码：`onclick="viewProject(''+p.name.replace(/'/g,"\\'")+'')"`
  - `''` 在单引号字符串内创建空字符串，导致 JS 解析器报 `SyntaxError: Unexpected string`
  - 整个 `<script>` 块因此无法执行，所有渲染函数都不会运行
- **修复方案**：改用 `data-idx` 属性传递项目索引（数字），避免字符串拼接
  - `renderProjects` 中：`filtered.map((p,i)=>{... onclick="viewProject("+i+")" ...})`
  - `viewProject(idx)` 中：通过 `el._filtered[idx]` 查找项目对象
- **验证**：`node --check` 对提取的 JS 代码通过语法检查
- **文件位置**：`/Users/rich/.openclaw/extensions/openclaw-lark/skills/poverty-alleviation-mgmt/scripts/pam.py`（第 740-758 行）
